Polytope

The word "polytope" means the generalization of a polygon or polyhedron in multiple dimensions. We can talk about the "cells" of a polytope, which are the generalization of sides or faces.

This program only looks at regular convex polytopes. That means all of the cells and angles are the same. In 3 dimensions, the five regular polytopes are called the Platonic Solids.

Types

Simplex

A simplex is the smallest polytope. A line segment, a triangle, and a tetrahedron are simlexes. In higher dimension you can make an n-simplex by taking an (n-1)-simplex and adding a new vertex above it connecting each of the existing vertices to the new one with a new edge.

There is a simplex in every dimension.

See Wikipedia's article on Simplex for more information.

Hypercube

A hypercube or sometimes called an n-cube is the generalization of a square or cube. There is a hypercube in every dimension.

See Wikipedia's article on Hypercube for more information.

Cross type

An n dimensional cross type polytope is made by taking an (n-1) dimensional cube and adding one point above and one point below the cube. Connect each of the cube's vertices to both of the new points. If you start with a square, you end up with an octohedron. If you start with a line segment, you end up with a square.

There is a cross type polytope in every dimension.

See Wikipedia's article on Cross-polytope for more information.

Dodecahedron

The Dodecahedron is only in 3 dimensions and has 12 faces. Each face is a pentagon.

See Wikipedia's article on Dodecahedron for more information.

Icosahedron

The Dodecahedron is only in 3 dimensions and has 20 faces. Each face is a triangle.

See Wikipedia's article on Icosahedron for more information.

24-cell

The 24-cell is only in 4 dimensions and has 24 cells. Each cell is an octohedron.

See Wikipedia's article on 24-cell for more information.
