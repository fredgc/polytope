Polytope

This program draws regular polytopes in several dimensions. See the overview for an explanation of what a polytope is. See controls for how to use the program.

Privacy and Data Retention

This program allows the user to change settings that are saved on your device. This is done via flutter's SharedPreferences which are backed by Android Shared Preferences on Android and LocalStorage for a web app. For Android, the data will be deleted when you uninstall the app. For the web, the data will be deleted when you clear application data in your browser.
